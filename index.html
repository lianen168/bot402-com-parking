What is the X402 Protocol?

The HTTP 402 "Payment Required" code has been a forgotten placeholder in the internet's foundation for decades. It was designed for a future that hadn't arrived: a native way for software to pay for digital resources without human intervention. Today, with digital currencies, this future is here. The X402 protocol is the realization of this vision—a way for any program to instantly request and receive a micropayment, creating a true machine-to-machine economy.
